# Beacon X1

Premium black-and-red DePIN website built with Next.js for Vercel deployment.

## Stack
- Next.js 15
- React 19
- TypeScript
- CSS (no external UI dependency required)

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. Import the repo into Vercel.
3. Framework preset: **Next.js**.
4. Build command: `next build`
5. Output setting: leave default.
6. Deploy.

## Included
- Premium landing page for **Beacon X1**
- Black and red brand styling
- 20 interactive demo surfaces
- Roadmap, FAQ, and CTA sections

## Notes
This project is intentionally self-contained so it is easy to deploy and customize.
